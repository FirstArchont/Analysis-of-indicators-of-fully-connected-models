# Анализ показателей полносвязных моделей

## Описание
- В этом проекте был проведён анализ показателей полносвязных моделей на датасетах MNIST и CIFAR-10
- Было создано множество графиков на Python + matplotlib
- Были созданы модели разной глубины и ширины
- Были реализованы модели с разными техниками регуляризации
